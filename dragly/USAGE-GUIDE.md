## 1. 소개

### drag.ly란?

drag.ly는 LLM(ChatGPT, Claude 등)과 대화하면서 떠오르는 생각을 임시로 저장해두었다가, 필요할 때 **드래그 & 드롭**으로 빠르게 삽입할 수 있는 macOS 메뉴바 유틸리티입니다.

### 왜 필요한가?

LLM 응답을 읽는 중에 새로운 질문이나 아이디어가 떠오르면:
- 바로 입력하면 현재 읽던 맥락이 끊김
- 다른 곳에 메모하면 나중에 복사/붙여넣기가 번거로움
- 머릿속에만 두면 잊어버리기 쉬움

**drag.ly는 이 문제를 해결합니다:**
1. 생각이 떠오르면 즉시 카드로 저장
2. 대화가 끝나면 카드를 드래그해서 바로 삽입
3. 사용한 카드는 자동으로 체크되어 정리 용이

### 핵심 특징

| 특징 | 설명 |
|------|------|
| **메뉴바 앱** | Dock에 표시되지 않아 화면 깔끔 |
| **Always-on-top** | 다른 창 위에 항상 표시 |
| **즉시 접근** | 글로벌 핫키로 어디서든 1초 내 접근 |
| **드래그 & 드롭** | 카드를 드래그해서 어디든 텍스트 삽입 |
| **데이터 유지** | 앱 재시작 후에도 카드 유지 |

## 2. 설치 및 실행

### 시스템 요구사항

- macOS 13.0 (Ventura) 이상
- Xcode 15.0 이상 (빌드 시)

### 설치 방법

#### 방법 1: Xcode에서 빌드

```bash
# 1. 프로젝트 폴더로 이동
cd dragly

# 2. Xcode로 프로젝트 열기
open dragly.xcodeproj

# 3. Xcode에서 ⌘+R 눌러서 빌드 및 실행
```

#### 방법 2: 터미널에서 빌드

```bash
cd dragly
xcodebuild -project dragly.xcodeproj -scheme dragly -configuration Release build
```

### 실행 확인

앱이 정상적으로 실행되면:
- **Dock**: 아이콘 없음 (의도된 동작)
- **메뉴바**: 스택 모양 아이콘 표시 ✓

```
┌────────────────────────────────────────────────┐
│ 🍎  파일  편집  보기  ...           📶 🔋 [≡]  │  ← 메뉴바에 아이콘
└────────────────────────────────────────────────┘
                                          ↑
                                    drag.ly 아이콘
```

## 3. 기본 사용법

### 3.1 윈도우 열기/닫기

| 방법 | 조작 |
|------|------|
| 마우스 | 메뉴바 아이콘 **좌클릭** |
| 키보드 | `⌃+⌥+D` (Control + Option + D) |
| 닫기 | `ESC` 키 또는 다시 클릭/단축키 |

### 3.2 카드 추가하기

```
┌─────────────────────────────────────┐
│ drag.ly                             │
├─────────────────────────────────────┤
│                                     │
│   (카드들이 여기에 표시됨)            │
│                                     │
├─────────────────────────────────────┤
│ Type a thought and press Enter... ⊕ │  ← 여기에 입력
└─────────────────────────────────────┘
```

1. 하단 입력창에 텍스트 입력
2. `Enter` 키 누르기
3. 새 카드가 상단에 추가됨

### 3.3 카드 드래그하기

```
┌─────────────────────────────────────┐
│ ≡  API rate limit 관련 질문      ✕  │  ← 이 카드를
└─────────────────────────────────────┘
         ↓ 드래그
┌─────────────────────────────────────┐
│ ChatGPT 입력창                      │  ← 여기에 드롭
│ ▌                                   │
└─────────────────────────────────────┘
```

1. 카드의 왼쪽 `≡` 핸들 또는 카드 전체를 드래그
2. 원하는 텍스트 입력 영역에 드롭
3. 텍스트가 자동으로 삽입됨
4. 카드가 **checked 상태**로 변경 (dim + shrink)

### 3.4 카드 정리하기

| 방법 | 설명 |
|------|------|
| 개별 삭제 | 카드에 마우스 올리면 나타나는 `✕` 버튼 클릭 |
| 일괄 삭제 | 헤더의 **"Clear Completed"** 버튼 클릭 |
| 전체 삭제 | 우클릭 메뉴 → **"Clear All"** |

## 4. 상세 기능

### 4.1 카드 상태

| 상태 | 외관 | 설명 |
|------|------|------|
| **일반** | 밝은 배경, 전체 높이 | 드래그 대기 상태 |
| **Checked** | 어두운 배경, 축소된 높이, 취소선 | 드래그 완료됨 |

```
일반 상태:
┌─────────────────────────────────────┐
│ ≡  이것도 물어봐야지               ✕ │  높이: 60px
└─────────────────────────────────────┘

Checked 상태:
┌─────────────────────────────────┐
│ ✓  이̶것̶도̶ ̶물̶어̶봐̶야̶지̶          ✕ │  높이: 32px, dim
└─────────────────────────────────┘
```

### 4.2 카드 편집

1. 카드를 **더블클릭**
2. 텍스트 수정
3. `Enter` 또는 바깥 클릭으로 저장

### 4.3 우클릭 메뉴

메뉴바 아이콘을 **우클릭**하면 컨텍스트 메뉴가 나타납니다:

```
┌─────────────────────────┐
│ Show / Hide             │  윈도우 토글
├─────────────────────────┤
│ Clear Completed         │  체크된 카드 삭제
│ Clear All               │  모든 카드 삭제
├─────────────────────────┤
│ Reset Window Position   │  기본 위치로 복원 *
├─────────────────────────┤
│ Quit drag.ly            │  앱 종료
└─────────────────────────┘

* 윈도우 위치/크기를 변경한 경우에만 표시
```

### 4.4 윈도우 위치/크기 기억

- 윈도우를 드래그하여 이동하면 위치가 저장됨
- 모서리를 드래그하여 크기 조절하면 크기가 저장됨
- 앱 재시작 후에도 마지막 위치/크기 유지
- 커스텀 위치일 때 메뉴바 아이콘이 변경되어 표시

### 4.5 데이터 영속성

- 모든 카드는 자동으로 저장됨
- 앱을 종료해도 카드 유지
- Mac 재시작 후에도 카드 유지

## 6. 사용 시나리오

### 시나리오 1: ChatGPT 대화 중 아이디어 저장

```
상황: ChatGPT가 긴 답변을 작성 중, 읽다가 추가 질문이 생각남

1. ⌃+⌥+D 눌러서 drag.ly 열기
2. "이 부분 더 자세히 설명해줘" 입력 → Enter
3. ESC로 닫고 계속 읽기
4. 또 생각나면 반복
5. 대화 끝나면 카드를 드래그해서 입력창에 드롭
```

### 시나리오 2: 코드 리뷰 중 피드백 수집

```
상황: PR 코드를 읽으면서 피드백 포인트 수집

1. drag.ly 열어두기
2. 코드 읽으면서 피드백 카드 추가:
   - "line 42: 변수명 더 명확하게"
   - "line 78: null 체크 필요"
   - "전체적으로 테스트 추가 필요"
3. 리뷰 작성할 때 카드들을 드래그해서 코멘트에 삽입
```

### 시나리오 3: 미팅 중 질문 정리

```
상황: 화상 미팅 중 질문할 내용 정리

1. 미팅 시작 전 drag.ly 열어두기
2. 발표 들으면서 질문 카드 추가
3. Q&A 시간에 카드 보면서 질문
4. 완료된 질문은 checked 처리
```

## 7. 설정 및 커스터마이징

### 7.1 윈도우 위치 변경

1. 윈도우 상단(타이틀바 영역)을 드래그
2. 원하는 위치로 이동
3. 자동 저장됨

### 7.2 윈도우 크기 변경

1. 윈도우 모서리를 드래그
2. 원하는 크기로 조절
3. 자동 저장됨
4. 최소 크기: 280 × 200 픽셀

### 7.3 기본 위치로 복원

1. 메뉴바 아이콘 **우클릭**
2. **"Reset Window Position"** 클릭
3. 화면 상단 중앙으로 복원

### 7.4 앱 아이콘 변경 (개발자용)

1. `Assets.xcassets/AppIcon.appiconset/`에 아이콘 이미지 추가
2. `Contents.json` 파일 수정
3. 앱 다시 빌드

## 8. 문제 해결

### Q: 글로벌 핫키 (⌃+⌥+D)가 작동하지 않아요

**A:** 손쉬운 사용 권한을 확인하세요:

1. **시스템 설정** 열기
2. **개인정보 및 보안** 클릭
3. **손쉬운 사용** 클릭
4. drag.ly 앱 찾아서 **체크 활성화**
5. 앱 재시작

### Q: 메뉴바에 아이콘이 안 보여요

**A:** 메뉴바 공간이 부족할 수 있습니다:

1. 메뉴바의 다른 아이콘들을 `⌘` 누른 상태로 드래그하여 정리
2. 또는 Bartender 같은 메뉴바 관리 앱 사용

### Q: 드래그한 텍스트가 삽입되지 않아요

**A:** 드롭 대상이 텍스트 입력을 지원하는지 확인하세요:

- **지원**: 브라우저 입력창, 텍스트 에디터, 메모 앱
- **미지원**: 이미지, 읽기 전용 영역, 일부 특수 입력 필드

### Q: 카드가 저장되지 않아요

**A:** 앱이 정상적으로 종료되었는지 확인하세요:

- 메뉴바 우클릭 → "Quit drag.ly"로 종료
- 강제 종료 시 마지막 상태가 저장되지 않을 수 있음

### Q: 윈도우가 다른 앱 뒤로 숨어요

**A:** 일부 전체 화면 앱에서는 floating 윈도우가 가려질 수 있습니다:

- 전체 화면 모드 대신 최대화 모드 사용
- 또는 `⌃+⌥+D`로 윈도우 다시 활성화
